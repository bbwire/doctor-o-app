# Doctor O Monorepo

## Quick Start

1. Install root dev tooling:

```bash
yarn install
```

2. Start API and PWA together:

```bash
yarn dev
```

## Run Individually

- API only: `yarn dev:api`
- PWA only: `yarn dev:pwa`
- Admin dashboard: `yarn dev:admin`

## Default Local URLs

- API: `http://127.0.0.1:8000`
- API base: `http://127.0.0.1:8000/api/v1`
- PWA: `http://localhost:3001`
- Admin: `http://localhost:3002`
